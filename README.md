# retail-sales-analysis

# Overview
This project dives into retail sales data to uncover trends, top-performing products, and how different regions contribute to revenue. 
The goal was to turn raw numbers into actionable business insights.

# Tools Used
Python (Pandas, Matplotlib, Seaborn) and Excel

# Process
Cleaned and structured the dataset  
Explored sales patterns and key metrics  
Visualized trends and regional performance  
Highlighted insights to guide business decisions

# Key Takeaways
Certain products and regions consistently outperform others  
Seasonal trends influence sales peaks and troughs  
Recommendations include targeted promotions and inventory adjustments

# Dataset
Simulated retail sales data including product details, sales numbers, and regional info.
